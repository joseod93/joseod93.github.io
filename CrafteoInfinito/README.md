# Crafteo Infinito

Juego de combinación de elementos inspirado en Infinite Craft, completamente en español.

Comienza con **4 elementos básicos** (Agua, Fuego, Tierra, Viento) y combínalos para descubrir **200+ elementos** organizados en cadenas temáticas: naturaleza, civilización, mitología, tecnología, comida y más.

## Cómo jugar

1. **Arrastra** un elemento del panel lateral al área de trabajo.
2. **Arrastra** un segundo elemento y suéltalo sobre el primero para combinarlos.
3. Si la combinación existe, aparecerá un nuevo elemento.
4. Los nuevos descubrimientos se añaden automáticamente al panel.
5. **Doble clic** en un elemento del área de trabajo para eliminarlo.
6. Usa el botón **Limpiar** para vaciar el área de trabajo, o **Reiniciar** para empezar desde cero.

Tu progreso se guarda automáticamente en el navegador.

## Despliegue en GitHub Pages

1. Sube este directorio a un repositorio de GitHub.
2. Ve a **Settings → Pages**.
3. En **Source**, selecciona la rama `main` y la carpeta `/ (root)`.
4. Haz clic en **Save**. El juego estará disponible en `https://<tu-usuario>.github.io/<nombre-repo>/`.

No requiere build, dependencias ni servidor — solo archivos estáticos.

## Tecnología

- HTML5, CSS3, JavaScript vanilla
- Zero dependencias (excepto Google Fonts vía CDN)
- Responsive: desktop y móvil
- Persistencia con `localStorage`
